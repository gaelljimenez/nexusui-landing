## 📋 Sobre o Projeto

O **NexusUI** é uma landing page conceitual que apresenta uma ferramenta fictícia de IA para geração de código e design systems. O projeto foi desenvolvido para demonstrar:

- ✅ **1 ID único** (`#hero-glow`) com estilização refinada
- ✅ **4 classes distintas** (`.badge-neon`, `.card-premium`, `.btn-prime`, `.grid-dashboard`)
- ✅ **1 seletor por atributo** (`[data-glow]`) com efeitos interativos
- ✅ **Box Model ajustado** (padding, margin, border, gap em todos os elementos)
- ✅ Design moderno com **glassmorphism**, gradientes e animações suaves
- ✅ **Totalmente responsivo** (mobile-first)
